This is a simple program for creating a playlist,
for practicing linked lists and iterators/list iterators.

The user can add albums and songs in the main method, then use the console
for the following actions:

0 - Quit
1 - Display instructions
2 - List my songs
3 - Show playlist
4 - Add song to playlist
5 - Play next song
6 - Play previous song
7 - Replay song
8 - Remove song from playlist
9 - Add all songs to playlist
